# QueryMind — Agentic Business Intelligence Assistant

An AI-powered data analysis agent that lets you chat with any CSV or Excel file in plain English.

## Live Demo
🔗 https://querymind-lclb.onrender.com

## Features
- Ask questions about your data in plain English
- Auto-generates and runs Pandas/SQL code
- Creates charts and visualizations instantly
- Remembers conversation context
- Detects business risks and anomalies

## Tech Stack
Python · FastAPI · LangChain · Groq LLaMA 3.3-70B · Pandas · Matplotlib
